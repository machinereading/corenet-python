# CoreNet Python API

## About
This repository is for CoreNet data, and Python API for using CoreNet. CoreNet is a Korean lexical smeantic network.

## prerequisite
* `python 3.5+`
* `pandas`
* `nltk`
* nltk 'wordnet' corpus (You can download it by `nltk.download('wordnet')`)

## How to use
refer `example.py`
